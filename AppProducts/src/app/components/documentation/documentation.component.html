Guía de Instalación y Ejecución del Proyecto

Instalación de Dependencias

Para comenzar, debes instalar las siguientes dependencias en tu proyecto Angular:

  1. PrimeNG: Biblioteca de componentes UI para Angular.
  2. PrimeFlex: Biblioteca de utilidades CSS para PrimeNG.
  3. Angular 16: Framework para desarrollar aplicaciones web.

Ejecuta los siguientes comandos en tu terminal para instalar las dependencias:

npm install primeng@^16.2.0 primeflex@^3.0.0

Nota: Asegúrate de tener Angular 16 ya instalado. Si no, puedes actualizarlo con:

ng update @angular/cli @angular/core

Ejecutar el Servidor de Angular

Para ejecutar el servidor de desarrollo y ver tu aplicación en acción, utiliza el siguiente comando:

ng serve

Esto iniciará el servidor de desarrollo en http://localhost:4200 de forma predeterminada.

Ejecutar Pruebas Unitarias

Para ejecutar las pruebas unitarias en tu proyecto, utiliza el siguiente comando:

ng test

Esto ejecutará las pruebas unitarias usando Karma y Jasmine. Asegúrate de que todas las pruebas estén correctamente configuradas para obtener resultados precisos.

Clonar el Proyecto desde GitHub

El proyecto está alojado en GitHub y puedes clonarlo desde la rama develop. Usa el siguiente comando para clonar el repositorio:

git clone -b develop https://github.com/IngMiguelDev/CrudProductos.git

Una vez clonado el repositorio, navega al directorio del proyecto y ejecuta el siguiente comando para instalar los paquetes de Node y las dependencias del proyecto:

npm install
